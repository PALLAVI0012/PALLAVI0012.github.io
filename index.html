Nishabdh - Farmer's Platform

Nishabdh connects farmers directly with consumers and offers services like OCR for document scanning, ASR for voice-based queries, and a multilingual interface. The platform provides a marketplace for buying and selling produce, access to agricultural advice, and various farmer support features. I designed the project's prototype using Figma, focusing on an intuitive user interface for seamless navigation. Currently under development, Nishabdh aims to streamline communication and improve farmers' livelihoods.
